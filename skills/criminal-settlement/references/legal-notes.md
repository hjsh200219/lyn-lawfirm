# 사건 유형별 법적 참고사항

합의서 작성 시 각 사건 유형별로 아래 법적 포인트를 확인하고, 필요 시 사용자에게 안내한다.

---

## 1. 상해·폭행 (형법 제257조, 제260조)

- **반의사불벌죄 여부**: 폭행죄(제260조)는 반의사불벌죄로, 피해자의 처벌불원 의사표시 시 공소권 없음 처리. 상해죄는 반의사불벌죄 아님 — 그러나 합의 시 공소 취소 또는 처벌 감경에 영향.
- **합의서 효력 발생 시점**: 합의금 수령과 고소 취하를 동시에 진행할 것을 권장.
- **주의**: 피해자가 합의금을 받은 후 고소 취하를 거부하는 사례 있음 → 공증 고려.
- **적정 합의금 참고**: 진단서 상 치료 기간, 실제 치료비, 위자료 합산.

## 2. 재물손괴 (형법 제366조)

- **반의사불벌죄**: 해당 없음 (친고죄도 아님).
- **손해 산정**: 수리비 견적서 또는 영수증 첨부 권장.
- **합의서에 포함할 것**: 손해 항목 명세, 수리비 + 위자료 구분 기재.
- **주의**: 차량 등 재물의 경우 감가상각 문제 발생 가능.

## 3. 명예훼손 (형법 제307조, 정보통신망법 제70조)

- **친고죄 여부**: 형법상 명예훼손은 반의사불벌죄. 정보통신망법상 명예훼손도 동일.
- **합의서 필수 포함**: 게시물 삭제 의무 + 이행 기한 + 이행 증빙 방법.
- **위약금 조항 강력 권장**: 게시물 재게시·추가 유포 방지.
- **허위 사실 vs 사실 적시**: 허위 사실인 경우 형사 처벌 가중 → 합의서에 허위 여부 기재 주의.

## 4. 사기·횡령 (형법 제347조, 제355조)

- **비친고죄**: 피해자 고소 없어도 수사 가능.
- **합의서 핵심**: 피해금액 전액 변제 + 고소 취하 + 추가 청구 포기.
- **분할 지급 시**: 지급 일정표 별지 첨부, 기한이익 상실 조항 추가 권장.
- **공증 강력 권장**: 사기 사건은 가해자 신뢰도 낮음 → 합의서 공증 필수.

## 5. 교통사고 (교통사고처리특례법, 형법 제268조)

- **보험 처리와 합의 병행**: 대인 보험 처리와 별개로 형사합의는 별도 진행.
- **합의서에 포함할 것**: 치료비·위자료·일실소득 포함 여부 명확히.
- **12대 중과실**: 신호위반, 중앙선 침범 등은 특례법 적용 제외 → 형사 처벌 가능.
- **주의**: 보험사 합의와 형사합의는 별개 — 보험 합의서에 형사합의 포기 조항 포함 여부 확인.

## 6. 성범죄

- **친고죄 폐지 (2013년 이후)**: 피해자 고소 없어도 수사·기소 가능.
- **합의서의 한계**: 형사합의가 공소 취소에 영향 없을 수 있음 → 검사 재량.
- **2차 피해 방지**: 합의 과정에서 피해자 정보 유출 금지 조항 필수.
- **합의 강요 절대 금지**: 합의 강요 시 추가 범죄(강요죄 등) 성립 가능.

---

## 합의서 공통 주의사항

1. **합의금 수령 전 고소 취하 절대 금지** — 합의금 수령과 동시 처리 원칙
2. **합의서 이중 원본 작성** — 피해자·가해자 각 1부 보관
3. **법원·수사기관 제출용은 공증 권장**
4. **합의서에 '이 사건과 관련한 모든 청구를 포기한다'는 표현** → 관련 민사소송도 차단 효과
5. **미성년자 가해·피해 시**: 법정대리인(부모) 서명 필수
