**운영 규모**: 변호사 1인. 팀 없음. 보조 인력 없음. 모든 검토·결재·법적 책임이 한 사람에게 집중.

**언어**: 프로젝트 산출물(plan, 문서, 커밋 메시지 본문, 사용자 노출 텍스트)은 한국어 기본. 코드/주석/기술 용어는 영어 허용. AGENTS.md, CLAUDE.md 등 메타 문서는 영어/한국어 혼용 허용.

**도메인**: 한국 법률 (변호사법, 형법, 민법, 형사절차). 외국법 자료는 부차적. 양형기준표·증거번호 체계·바탕체 12pt A4 포맷 등 한국 실무 규칙 준수 필수.

**운영 제약이 설계에 미치는 영향**:
- 변호사 검토 게이트는 옵션이 아닌 필수 — 우회 경로 설계 금지
- 자동화의 목표는 "검토 시간 단축"이지 "검토 생략"이 아님
- 비용·시간 트레이드오프에서 "사람 1명 시간"이 가장 비싼 자원
- 동시 다중 사건 처리량보다 사건당 품질·책임 추적성 우선
- 별도 서버·인프라 운영 부담을 견딜 수 없음 (Claude Code 플러그인 + MCP가 최대 복잡도)

**How to apply:**
새 기능/스킬 제안 시 다음 3가지 자동 체크:
1. "변호사 1명이 검토 가능한 산출 빈도인가" — 사건당 토큰·시간 예산 고려
2. "검토 게이트가 명시적으로 존재하는가" — soft gate라도 우회 경로 노출 금지
3. "한국어 출력이 기본인가" — 사용자 노출 텍스트 점검

이영남 변호사의 연락처·CTA 정보는 `references/cta-config.md`가 단일 source of truth.
