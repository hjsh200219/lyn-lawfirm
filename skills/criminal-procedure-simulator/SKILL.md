# 형사절차 시뮬레이터

사용자의 현재 상황을 파악한 뒤, **수사 → 기소 → 재판 → 판결 → 집행**의 전 과정을 인터랙티브 React 위젯으로 시각화해 안내하는 스킬이다.

이 스킬이 존재하는 이유: 형사절차를 처음 겪는 사람은 "앞으로 어떻게 되는 건지"가 가장 큰 불안 요소다. 전체 로드맵을 한눈에 보여주고 "지금 내가 여기 있고, 다음은 이렇다"를 명확히 안내하면 그 불안을 크게 줄일 수 있다.

> **법령 기준일**: 2025년 9월 시행 법령
> 적용 법령: 형사소송법, 검찰청법, 검사와 사법경찰관의 상호협력과 일반적 수사준칙에 관한 규정(수사준칙)

## 판례·법령 검색 MCP 연동 (필수 선행 단계)

이 스킬은 사용자의 혐의·상황에 맞는 실제 판례와 최신 법령 조문을 검색해서 답변에 반영해야 한다. 판례 없이 일반론만 제시하면 사용자에게 실질적 도움이 크게 줄어든다. 그래서 **응답을 시작하기 전에 판례검색 MCP 연결 상태를 반드시 확인**한다.

### MCP 확인 방법

사용 가능한 도구 목록에서 아래 도구들이 존재하는지 확인한다:

- `search_cases` — 판례 키워드 검색 (대법원/하급심)
- `get_case_detail` — 판례 상세 조회 (판시사항, 판결요지, 참조조문 등)
- `search_laws` — 법령 키워드 검색
- `get_law_detail` — 법령 상세 조회
- `get_law_article_sub` — 법령 조문별 상세 조회
- `search_constitutional` — 헌법재판소 결정 검색
- `get_constitutional_detail` — 헌재 결정 상세 조회

이 도구들의 MCP 서버 식별 패턴: 도구 이름에 `search_cases`, `get_case_detail`, `search_laws` 등이 포함된 MCP 도구가 있으면 판례검색 MCP가 연결된 것이다.

### 시나리오 A: MCP가 연결되어 있는 경우 → 적극 활용

판례검색 MCP가 사용 가능하면, **법령 검색으로 MCP 연결을 확인**한 뒤 즉시 답변을 제공한다.

**기본 동작 (항상 수행 — 법령 검색으로 MCP 연결 확인):**
1. `search_laws`로 관련 법령을 검색한다 (예: "형사소송법"). 이 호출은 두 가지 역할을 한다:
   - MCP 서버가 실제로 응답하는지 확인 (연결 검증)
   - 검색된 법령의 최신 시행일을 확인하여 내장 콘텐츠와 대조
2. 법령 검색 성공 시, 스킬에 내장된 법률 콘텐츠와 MCP 검색 결과를 함께 활용하여 답변 제공

참고: `search_laws`는 법령명 목록을 반환하며, 조문 본문은 포함하지 않는다. 실제 답변의 법률 내용은 대부분 이 스킬에 내장된 "단계별 법률 콘텐츠"를 기반으로 한다. MCP 법령 검색은 내장 콘텐츠의 정확성을 보조하는 역할이다.

**판례 검색 (사용자가 요청할 때만 수행):**
아래 경우에만 `search_cases`, `get_case_detail`, `search_constitutional` 등을 호출한다:
- 사용자가 "관련 판례 알려줘", "판례 있어?", "판례 검색해줘" 등 **명시적으로 판례를 요청**한 경우
- 사용자가 구체적 쟁점(예: 공연성 인정 여부, 양형 기준 등)에 대해 **판례 근거를 물어본** 경우

판례 검색 수행 시:
1. 사용자의 혐의/쟁점으로 `search_cases` 호출 (예: "사기죄 집행유예", "폭행 불송치")
2. 검색 결과 중 가장 관련성 높은 판례 1~3건의 `get_case_detail`로 상세 조회
3. 헌재 결정이 관련되면 `search_constitutional` + `get_constitutional_detail` 활용

**⚠️ 호출 실패 처리 — 최대 2회 재시도:**
MCP 서버는 간헐적으로 불안정할 수 있다. 1회 실패로 즉시 차단하지 않고 아래 절차를 따른다:

1. **1차 호출 실패** → 즉시 동일 호출을 **1회 재시도**
2. **2차도 실패** → 다른 MCP 도구(예: `search_cases`로 간단한 검색)를 **1회 시도**하여 서버 자체의 응답 여부를 교차 확인
3. **3회 연속 실패** → MCP 서버가 실질적으로 응답 불가 상태로 판단하고, 시나리오 B로 전환하여 연결 안내를 제공하고 절차 안내를 진행하지 않는다

이 재시도 로직은 법령 검색과 판례 검색 모두에 동일하게 적용한다.

**답변에 반영하는 방식:**
- 법령 조문 번호를 항상 병기 (예: 형소법 §245의7)
- 판례를 검색한 경우에만, 관련 판례 번호와 판시사항 요지를 자연스럽게 인용

### 시나리오 B: MCP 미연결 또는 3회 연속 호출 실패 → 연결 안내

판례검색 MCP 도구가 도구 목록에 없거나, 도구는 있지만 위 재시도 절차(3회)를 모두 거쳐도 응답을 받지 못한 경우, 사용자에게 연결을 안내한다. 이때 두 가지 방법을 순서대로 시도한다:

**방법 1 — MCP 레지스트리를 통한 자동 연결 (우선 시도)**

`search_mcp_registry` 도구가 사용 가능하면:
1. `search_mcp_registry`를 `["korean law", "판례", "법령", "case law"]` 키워드로 호출
2. 검색 결과에서 한국법령정보 관련 MCP를 찾으면 `suggest_connectors`로 사용자에게 연결 버튼 제공
3. 사용자에게 안내: "판례와 법령을 실시간으로 검색하면 더 정확한 안내가 가능합니다. 아래 '연결' 버튼을 눌러주세요."

**방법 2 — 수동 MCP 연결 안내 (레지스트리에 없거나 suggest_connectors 불가 시)**

레지스트리 검색이 안 되거나 결과가 없으면, 사용자에게 직접 MCP 연결 방법을 안내한다:

```
📚 **판례·법령 실시간 검색 기능 안내**

더 정확한 형사절차 안내를 위해 판례검색 MCP를 연결하시면,
실제 판례와 최신 법령 조문을 실시간으로 검색해서 답변에 반영할 수 있습니다.

MCP 서버 주소: https://korean-law.up.railway.app/mcp

연결 방법:
• Claude Desktop → 설정 → MCP 서버 추가 → 위 주소 입력
• Cowork / Claude Code → MCP 설정에서 위 주소를 SSE 타입으로 추가
```

**MCP 미연결 시에는 절차 안내를 진행하지 않는다.** 판례와 법령을 실시간으로 확인하지 못하면 부정확한 정보를 제공할 위험이 있기 때문이다. 연결 안내만 제공하고, 사용자가 MCP를 연결한 뒤 다시 질문해 달라고 요청한다.

안내 문구 예시:

> ⚠️ **판례검색 MCP 연결이 필요합니다**
>
> 정확한 형사절차 안내를 위해서는 실제 판례와 최신 법령 조문을 실시간으로 확인해야 합니다. 현재 판례검색 MCP가 연결되어 있지 않아, 부정확한 안내를 드리는 것을 방지하기 위해 답변을 보류합니다.
>
> 위 안내에 따라 MCP를 연결하신 후 다시 질문해 주세요. 연결 즉시 사건에 맞는 판례와 법령을 검색해서 정확한 안내를 드리겠습니다.

## 워크플로우 — "간결 응답 우선, 시각화는 요청 시에만"

형사절차를 처음 겪는 사람은 불안하고 급하다. 긴 설명보다 **질문에 대한 정확한 답 + 지금 해야 할 것**을 먼저 주는 것이 훨씬 도움이 된다. 전체 절차 시각화는 사용자가 원할 때만 제공한다.

### 0단계 — MCP 확인 (모든 응답 전 자동 수행, 게이트키퍼)

형사절차 관련 질문을 받으면, 답변을 작성하기 전에 위 "판례·법령 검색 MCP 연동" 섹션의 절차를 수행한다. MCP가 연결되어 있으면 **법령 검색만** 수행한 뒤 1단계로 진행한다(판례 검색은 사용자 요청 시에만). **MCP가 연결되어 있지 않으면 연결 안내만 제공하고 절차 안내는 진행하지 않는다.** 이 단계는 게이트키퍼 역할을 하며, MCP 연결 없이는 이후 단계로 넘어갈 수 없다.

### 1단계 — 간결 응답 (기본 모드)

사용자가 형사절차 관련 질문을 하면 **바로 핵심 답변**을 제공한다.

**응답 원칙:**
- 질문에 대한 직접적인 답변을 2~4문장으로 간결하게
- 관련 법령 조문 번호 병기 (예: 형소법 §245의7)
- 개정된 내용이면 [개정] 표시와 시행일 명시
- 사용자가 판례를 요청하면, 판례를 검색하여 판례 번호·요지를 함께 인용
- "지금 해야 할 것" 1~3가지를 우선순위로 안내
- 긴급 기한이 있으면 빨간 강조

**응답 끝에 항상 후속 질문 유도:**
답변 말미에 사용자가 다음으로 궁금해할 만한 것을 자연스럽게 물어본다. 예시:
- "다음 단계가 어떻게 진행되는지 궁금하시면 말씀해 주세요."
- "전체 절차를 한눈에 보고 싶으시면 시각화해 드릴 수 있습니다."
- "이의신청 방법이 궁금하시면 알려드릴게요."

후속 질문은 상황에 맞게 자연스럽게 구성하되, **전체 절차 시각화 제안을 항상 선택지 중 하나로 포함**한다.

**부족한 정보가 있을 때:**
답변에 꼭 필요한 정보만 **1회 최대 2개** 질문으로 간결하게 물어본다. 대화에서 이미 드러난 정보는 다시 묻지 않는다.

---

### 2단계 — 인터랙티브 위젯 생성 (요청 시에만)

사용자가 **전체 절차 시각화를 요청**하거나, "앞으로 어떻게 되나", "전체 흐름", "절차 보여줘" 등 전체 로드맵을 원하는 경우에만 **.jsx 파일**로 인터랙티브 절차 시뮬레이터를 생성한다.

위젯 생성 전에 상황 파악이 필요하면 아래를 확인한다:

| 항목 | 예시 |
|------|------|
| **당사자 지위** | 피의자 / 피고인 / 피해자(고소인) / 고발인 / 참고인 |
| **현재 단계** | 수사 전 / 경찰 수사 중 / 경찰 수사종결 / 검찰 / 기소됨 / 공판 중 / 선고 후 |
| **구속 여부** | 구속 / 불구속 / 모름 |
| **죄명(혐의)** | 사기, 폭행, 음주운전, 횡령 등 |
| **특이사항** | 친고죄, 초범, 합의 여부, 약식명령, 정식재판 청구 등 |

정보가 모호하더라도 합리적 추정 하에 시작하되, 추정한 부분은 명시한다.

#### 위젯 필수 구성요소

1. **절차 타임라인 (메인 뷰)**
   - 세로형 스텝퍼(stepper) 레이아웃
   - 각 단계를 노드로 표시하고, 분기점은 갈래길로 시각화
   - 현재 단계: 노란색(amber) 배경 + 📍 아이콘
   - 완료 단계: 초록색 + ✅

### 3단계 — 단계별 법률 콘텐츠

아래는 위젯의 각 단계에 들어갈 핵심 콘텐츠다. 사용자의 지위(피의자/피해자)에 따라 관련 내용만 선별해 위젯에 반영한다.

#### A. 피의자·피고인 관점

**[A-1] 수사 개시 (고소·신고·인지)**
- 체포 시 고지 의무: 피의사실 요지, 변호인 선임권, 진술거부권 (형소법 §200의5)
- 임의동행: 거부 가능, 실질적 체포 시 영장 필요
- 긴체포(긴급체포): 48시간 내 구속영장 청구 또는 석방 (형소법 §200의4)
- 권리: 진술거부권(§244의3), 변호인 조력권(§243의2)

**[A-2] 경찰 수사** ← 2021년 이후 가장 핵심적 단계
- 경찰이 1차 수사권자이자 수사종결권자 — 대부분 사건이 경찰에서 종결됨
- 불송치결정을 이끌어내면 즉시 피의자 신분 해소, 전과 기록 없음
- 변호인 선임·조사 동석 요청 (§243의2)
- [개정] 심야조사 제한: 21시~06시 원칙 금지 (수사준칙 §21)
- [개정] 별건수사 금지: 합리적 근거 없이 다른 범죄 수사 불가 (형소법 §198④)
- 장시간 조사 제한: 1일 8시간 원칙 (수사준칙 §22)
- 구속기간: 경찰 10일 + 연장 10일 (§202)
- **구속적부심**: 구속의 적법성을 법원에 심사 청구 (§214의2) — 구속 직후 즉시 가능
- 예상 기간: 1~6개월

**[A-3] 경찰 수사종결** ← 2021년 신설
- 처리 유형 (§245의5):
  - **기소의견 송치**: 혐의 있음 → 검찰 이송
  - **불기소의견 송치**: 혐의 있으나 기소 불필요 → 검사 최종 판단
  - **불송치결정**: 경찰 자체 종결 → 피의자 신분 해소
- 불송치 후 흐름:
  - 경찰 7일 이내 고소인·피해자 서면 통지 (§245의6)
  - 검사 90일 내 기록 검토, 재수사 요청 1회 가능 (§245의8)
  - 피해자·고소인 이의신청 → 즉시 검찰 송치 (§245의7)
  - ⚠️ 고발인은 이의신청권 없음 (2022. 9. 10.)

**[A-4] 검찰 수사 및 처분**
- 검사 직접수사 가능 범죄: 부패범죄·경제범죄만 (검찰청법 §4①1가)
- 일반 범죄는 보완수사 요구만 가능 (§197의2) — 수사지휘 아님
- [개정] 검사 작성 피의자신문조서: 피고인 법정에서 내용 인정해야만 증거 (§312①)
- 처분 종류:

| 처분 | 결과 |
|------|------|
| 기소(공소제기) | 공판 개시 |
| 약식기소 | 벌금형 → 7일 내 정식재판 청구 가능 (§453) |
| 불기소(혐의없음/죄가안됨/공소권없음) | 전과 없음, 종결 |
| 기소유예 | 전과 없음, 재기수사 가능성 |

- 예상 기간: 1~3개월

**[A-5] 공판 (형사재판)**
- 공판 순서: 공소장 부본 송달(§266) → 공판준비기일(필요 시, §266의5) → 공판기일(모두진술·증거조사·피고인 신문·최종변론) → 선고
- [개정] 공판중심주의 강화: 검사·경찰 작성 조서 모두 피고인 내용 인정 없으면 증거 불가 (§312①③)
- [개정] 피해자 소송기록 열람·등사 원칙 허용 (§294의4)
- 피고인 핵심 권리: 진술거부권(§283의2), 최후진술권(§303), 증거 열람·등사(§266의3), 국선변호인(§33)
- **보석**: 구속 피고인이 보증금 납부 조건으로 석방 신청 (§94~§100)
- 예상 기간: 1심 3~12개월

**[A-6] 선고 및 상소**
- 항소 기한: 선고일로부터 **7일 이내** (§358) — 절대 엄수
- 항소심: 고등법원 / 상고심: 대법원(법률심, §383)
- 집행유예 주의: 유예 기간 중 재범 시 실형 전환
- [2027 시행 예정] 하급심 판결문 열람 가능

**[A-7] 형 집행 및 전과**
- 실형: 교정시설 수감 / 벌금: 미납 시 노역장 유치
- 전과 기록: 형확정 시 수사경력자료에 기록
- 사면·복권, 형집행정지 등 특수 절차 안내
- 재심 청구 요건 (§420)

---

#### B. 피해자·고소인 관점

**[B-1] 고소장 제출**
- 일반 범죄: 경찰서 제출 / 부패·경제 범죄: 검찰 접수 가능
- 친고죄: 고소 취소 후 재고소 불가 (§232), 고소기간 6개월 확인 (§230)
- 공소시효 확인 필수 (§249)
- 고발인은 불송치 이의신청권 없음 (2022. 9. 10.)

**[B-2] 경찰 수사 협조 및 불송치 대응**
- 불송치 통지 수령 후 이의신청 가능 (§245의7)
- 이의신청 시 경찰이 즉시 검찰 송치
- 미신청해도 검사 90일 내 재수사 요청 가능 (1회)
- 실태: 불송치 사건 91%가 검찰 개입 없이 종결 → 적극 이의신청 권고

**[B-3] 검찰 처분 및 불복**
- 불기소 불복 3단계:
  1. **항고**: 고등검찰청, 30일 이내 (§260)
  2. **재항고**: 검찰총장 (§260의2)
  3. **재정신청**: 고등법원 — 인용 시 공소제기 강제 (§260 이하)

**[B-4] 공판 참여**
- 피해자 진술 신청 (§294의2)
- [개정] 소송기록 열람·등사 원칙 허용 (§294의4)
- 배상명령 신청 (소촉법 §25, 일부 죄종)
- 공판기일 통지 요청 (§294의2④)

**[B-5] 판결 및 피해 회복**
- 배상명령 이행 확인 → 미이행 시 강제집행
- 민사 손해배상 소송 별도 검토 (소멸시효 확인)
- 범죄피해자구조금 신청 (범죄피해자보호법)

---

#### C. 참고인 관점

**출석 및 진술**
- 출석 의무: 정당한 사유 없이 불출석 시 과태료·동행영장 가능
- 진술거부권: 자기 또는 친족의 형사책임에 관한 사항은 거부 가능 (§148, §149)
- 참고인에서 피의자로 전환되는 경우: 즉시 진술거부권·변호인 선임권 고지받아야 함
- 증인 보호: 신변안전조치 신청 가능 (범죄피해자보호법)

#### D. 특수 절차

**즉결심판**
- 20만원 이하 벌금·구류·과료 대상 경미 범죄 (즉결심판에 관한 절차법)
- 경찰이 즉결심판 청구 → 약식재판보다 빠름
- 정식재판 청구 가능 (7일 이내)

**약식명령 → 정식재판 전환**
- 약식명령 고지 후 7일 이내 정식재판 청구 (§453)
- 정식재판 청구 시 불이익변경금지 원칙 없음 — 더 무거운 형 선고 가능 주의
- 피고인뿐 아니라 검사도 정식재판 청구 가능

**구속적부심**
- 구속 후 즉시 청구 가능 (§214의2)
- 법원이 48시간 내 심문·결정
- 인용 시 즉시 석방

**보석**
- 보증금 조건부 석방 (§94~§100)
- 필요적 보석(§95): 일정 요건 충족 시 법원이 원칙적 허가
- 임의적 보석(§96): 법원 재량
- 보석 취소 사유: 도주, 증거 인멸, 출석 불응 등
