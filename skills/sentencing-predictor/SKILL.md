# 양형예측(量刑豫測) 스킬

## 개요

형사사건에서 예상 선고형을 분석하는 스킬이다. 대법원 양형위원회의 양형기준표, 실제 판례 검색 결과, 특별·일반 양형인자를 종합하여 감경/기본/가중 구간 중 어디에 해당하는지 판단하고, 시나리오별 형량과 집행유예 가능성을 예측한다.

양형 예측은 100% 정확할 수 없다. 실제 선고는 재판부의 재량에 따르며, 이 스킬은 양형기준과 유사 판례를 기반으로 한 합리적 추정을 제공하는 것이다. 이 한계를 사용자에게 반드시 고지한다.

---

## Step 1: 사건 정보 수집

### 1-A. 대화 내역에서 기존 정보 추출

사용자가 이전 대화에서 이미 언급한 정보(죄명, 피해액, 전과 여부 등)는 다시 묻지 않는다. 대화 내역에서 추출할 수 있는 정보를 최대한 채우고, 빠진 항목만 질문한다.

### 1-B. AskUserQuestion으로 핵심 정보 수집

빠진 정보를 AskUserQuestion 도구를 활용하여 선택지 형태로 수집한다. 한 번에 최대 4개 질문까지 묶을 수 있으므로, 필수 항목을 효율적으로 나누어 질문한다.

**반드시 수집해야 할 항목:**

```
[기본 사건 정보]
- 죄명 (사기, 폭행, 절도, 횡령, 배임, 성범죄, 마약, 교통사고 등)
- 피해액 또는 범행 규모 (재산범죄의 경우)
- 범행 횟수 (단일범행 / 다수범행·상습)

[피고인 정보]
- 전과 유무 (초범 / 동종전과 / 이종전과)
- 자백 여부 (전부 자백 / 일부 자백 / 전부 부인)
- 현재 구속 여부

[피해자 관련]
- 피해자 유형 (일반인 / 고령자·취약계층 / 불특정 다수 / 신뢰관계 이용)
- 피해 변제 상황 (전액 변제 / 일부 변제(비율) / 변제 없음)
- 피해자 합의·처벌불원 여부

[특별사정]
- 조직적·계획적 범행 여부
- 범행 수법의 특이성 (보이스피싱, 투자사기, 보험사기 등)
- 기타 특별양형인자 (자수, 공범 여부, 사회봉사 등)
```

모르는 항목이나 답하기 어려운 항목은 "미확인"으로 처리하고 진행한다. 완벽한 정보를 기다리지 않는다.

### 1-C. 재산범죄의 피해액 구간 확인

재산범죄(사기, 횡령, 배임, 절도 등)는 피해액이 양형의 핵심 변수이므로, 구체적인 금액 구간을 반드시 확인한다.

**사기죄 기준 유형 분류:**
| 유형 | 피해액 범위 |
|------|-----------|
| 1유형 | 1백만 원 이하 |
| 2유형 | 1백만 원 초과 ~ 5천만 원 이하 |
| 3유형 | 5천만 원 초과 ~ 5억 원 이하 |
| 4유형 | 5억 원 초과 ~ 30억 원 이하 |
| 5유형 | 30억 원 초과 |

> 피해액 5억 원을 초과하면 특정경제범죄가중처벌법 적용 가능성이 있으므로 별도 안내한다.

## Step 2: 양형기준표 조회

### 2-A. 대법원 양형위원회 사이트 접속

WebFetch를 이용하여 대법원 양형위원회 사이트에서 해당 죄명의 양형기준을 조회한다.

**주요 URL 구조:**
- 메인: `https://sc.scourt.go.kr/sc/krsc/main/Main.work`
- 사기죄: `https://sc.scourt.go.kr/sc/krsc/criterion/criterion_10/fraud_01.jsp`

다른 죄명의 경우, 메인 페이지에서 해당 범죄 유형의 링크를 먼저 찾은 뒤 상세 페이지에 접속한다.

WebFetch prompt 예시:
```
"이 페이지에서 [죄명] 양형기준의 전체 내용을 추출해주세요.
유형 분류, 감경/기본/가중 각각의 형량 범위, 특별양형인자(감경요소, 가중요소),
일반양형인자, 집행유예 기준을 원문에 가깝게 모두 추출해주세요."
```

### 2-B. 양형기준표 정리

조회 결과를 아래 형태의 표로 정리한다:

```
[양형기준 - ○○죄 제○유형]

| 구간 | 권고 형량 |
|------|---------|
| 감경 | ○년○월 ~ ○년○월 |
| 기본 | ○년○월 ~ ○년○월 |
| 가중 | ○년○월 ~ ○년○월 |
```

## Step 3: 판례 검색

### 3-A. 판례검색 MCP 활용

`search_cases` 도구로 유사 사건 판례를 검색한다. 검색어는 핵심 키워드를 2~3개 조합한다.

**검색 전략:**
1. 죄명 + 피해액 규모 + 핵심 양형인자 조합으로 검색
2. 검색 결과가 없으면 키워드를 줄이거나 변경하여 재검색
3. 유관한 판례가 나오면 `get_case_detail`로 상세 내용 확인

**검색 예시:**
```
search_cases(query="사기 피해액 1억 초범 자백")
search_cases(query="사기죄 형법 제347조 양형")
search_cases(query="횡령 집행유예 피해변제")
```

### 3-B. 법령 조문 확인

`search_laws` → `get_law_article_sub`로 적용 법령의 정확한 조문을 확인한다.

**주요 확인 조문:**
- 해당 죄명의 법정형 (예: 형법 제347조 사기죄 — 10년 이하 징역 또는 2천만 원 이하 벌금)
- 형법 제62조 (집행유예 요건 — 3년 이하 징역·금고 선고 시 가능)
- 특정경제범죄가중처벌법 (피해액 5억 초과 시)
- 특별법 적용 여부 (성폭력처벌법, 교통사고처리특례법 등)

---

## Step 4: IRAC 양형 분석

수집한 양형기준, 판례, 법령을 종합하여 IRAC 구조로 분석한다.

### Issue (쟁점)
"이 사건에 감경 구간이 적용되는가, 기본 구간이 적용되는가, 가중 구간이 적용되는가?"

### Rule (법규범)
양형위원회 양형기준에 따르면:
- 특별감경인자가 특별가중인자보다 우세하면 → **감경 영역**
- 양자가 균형이면 → **기본 영역**
- 특별가중인자가 우세하면 → **가중 영역**

### Application (적용)
사용자의 사건에 존재하는 양형인자를 감경 방향과 가중 방향으로 나누어 분석한다.

**감경 방향 인자** — 아래 중 해당하는 것을 체크:
- 초범 (일반감경)
- 자백 + 반성 (일반감경)
- 피해 전액 변제 (특별감경)
- 피해 일부 변제 (일반감경)
- 피해자 합의·처벌불원 (특별감경)
- 자수 (특별감경)
- 범행 가담 정도가 경미 (일반감경)
- 우발적 범행 (일반감경)

**가중 방향 인자** — 아래 중 해당하는 것을 체크:
- 동종전과 (특별가중)
- 조직적·상습적 범행 (특별가중)
- 취약계층 대상 (특별가중)
- 신뢰관계 이용 (특별가중)
- 다수 피해자 (일반가중)
- 범행 수법 교묘 (일반가중)
- 피해 회복 노력 없음 (일반가중)
- 범행 후 도주·증거인멸 (일반가중)

### Conclusion (결론)
감경·가중 인자의 우열을 판단하여 적용 구간과 그 구간 내 예상 형량을 결정한다.

### 시나리오 설계 원칙

시나리오는 사건 유형과 현재 상태에 따라 달라진다. 핵심은 "사용자가 지금부터 할 수 있는 행동"에 따른 결과 차이를 보여주는 것이다.

**재산범죄 (사기, 횡령, 배임 등):** 변제 비율과 피해자 합의가 핵심 변수
- 현 상태 유지
- 추가 변제 (70% 이상) + 피해자 합의
- 전액 변제 + 처벌불원서 확보

**폭력범죄 (폭행, 상해 등):** 피해자 합의와 피해 정도가 핵심
- 현 상태 유지
- 피해자 합의 + 처벌불원서
- 합의 + 추가 정상자료 (반성문, 탄원서, 사회봉사)

**교통사고:** 보험 처리, 피해자 합의, 음주·무면허 여부
**성범죄:** 피해자 합의 효과 제한적, 반성·치료 프로그램 수강 여부
**마약:** 초범 여부, 투약량, 자수·자백, 치료 의지

### 집행유예 판단 기준

형법 제62조에 따라 집행유예의 법적 요건:
- **3년 이하의 징역 또는 금고** 선고 시 가능
- 정상에 참작할 사유가 있을 때
- 형을 선고한 전력이 없거나, 집행유예 이상의 전과가 없을 때

**집행유예 가능성 평가 기준표:**

| 가능성 | 비율 | 주요 조건 |
|--------|------|----------|
| 매우 높음 | 80~95% | 초범 + 전부자백 + 전액변제 + 처벌불원서 |
| 높음 | 65~80% | 초범 + 자백 + 상당 변제(70%↑) + 합의 |
| 보통 | 45~65% | 초범 + 자백 + 일부 변제(30~70%) |
| 낮음 | 25~45% | 변제 미미 + 합의 없음, 또는 동종전과 있음 |
| 매우 낮음 | 10~25% | 동종전과 + 변제 없음 + 부인, 또는 예상형 3년 초과 |

이 표는 대략적인 기준이며, 실제 사건의 구체적 사정에 따라 달라진다. 사용자에게 제시할 때는 "약 ○○~○○%" 형태로 범위를 주고, 그 근거를 함께 설명한다.
